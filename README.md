Sistema de Gerenciamento Veterinário

Sobre o Projeto

Este sistema foi desenvolvido como parte da avaliação P2 da disciplina de Programação Orientada a Objetos.
Ele implementa um gerenciamento completo de uma clínica veterinária, incluindo cadastro de proprietários e animais, agendamentos, cancelamentos, remarcações e relatórios de eventos médicos.

O projeto utiliza Java, aplicando princípios essenciais da POO:

Herança

Classe abstrata

Polimorfismo

Encapsulamento

Composição

ArrayList para armazenamento de dados

O programa funciona totalmente via console.

COMO EXECUTAR

Baixe ou clone o repositório.

Abra a pasta do projeto na sua IDE.

Certifique-se de que todos os arquivos .java estão no mesmo pacote.

Execute o arquivo:

Main.java

O menu principal será exibido automaticamente.
